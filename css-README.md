# CSS Review

## HTML Tag Selector
```
body {
    background-color: blanchedalmond;
    font-family: monospace;
    text-align: center;
}
```

## Class Selector
```
.container{
    border: solid wheat;
    height: 200px;
    width: 300px;
    margin: auto;
}   
```

## Hover Effect
```
.container:hover{
    background-color: wheat;
}
```
